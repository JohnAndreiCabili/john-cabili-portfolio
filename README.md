## Getting Started / Installation

To run this portfolio locally:

```bash
# Clone the repo
git clone https://github.com/JohnAndreiCabili/john-cabili-portfolio.git

cd john-cabili-portfolio

# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
